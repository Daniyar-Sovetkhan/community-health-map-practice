# Contributing to Community Health Map

Thank you for your interest in contributing! 🎉

## How You Can Help

- **Report bugs** by creating an Issue
- **Suggest features** by creating an Issue
- **Improve documentation** by submitting a Pull Request
- **Write code** by fixing issues labeled `good first issue`

## Getting Started

1. **Fork** the repository to your GitHub account
2. **Clone** your fork locally
3. **Create a branch** for your changes
4. **Make your changes** and commit them
5. **Push** to your fork
6. **Open a Pull Request** with "Closes #[issue number]" in the description

## Code of Conduct
Please read our [Code of Conduct](CODE_OF_CONDUCT.md) before contributing.
