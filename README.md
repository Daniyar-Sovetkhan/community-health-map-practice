# Community Health Map
An open-source web application that allows communities to anonymously report and visualize local health trends (like flu symptoms or seasonal allergies) to help residents and local organizations make informed decisions.
### Team contributor: Serik Ersultan
Update README.md
## Problem Statement

In many communities, people have no way to know about local health trends. If many neighbors have flu symptoms, there is no alert system. Schools don't know when to close, clinics don't know when to prepare.

## Our Solution

Community Health Map provides:
- Anonymous symptom reporting
- Real-time map visualization
- Optional email alerts for local outbreaks

## SDG Alignment

This project contributes to **SDG 3: Good Health and Well-being** by enabling early detection of local health trends and empowering communities to respond proactively.

## Key Features

- **Community Map** - See reported symptoms in your area
- **Anonymous Reports** - No personal data collected
- **Alert Subscription** - Get notified when symptoms exceed normal levels
- **Privacy by Design** - No PII stored
